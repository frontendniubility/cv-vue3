Curriculum Vitae (CV)

* Made with Vue.js v3
* Easy to build, export and host anywhere you like
* Exports nicely to PDF from the browser
* Custimize to your own needs!


Do this:

```sh
> git clone git@codeberg.org:fractalf/cv-vue3.git
> cd cv-vue3
> npm i
> npm run dev
```

..and you get this!
